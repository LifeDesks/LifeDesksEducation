INSTALLATION
------------

1. Place the "og_content_permissions" folder in your modules directory e.g. "sites/all/modules".
2. Enable the module under Administer >> Site building >> Modules (admin/build/modules).
3. Grant permission "administer og content permissions" to roles that should be able to control
   which OG content node types should have access control applied.
4. Go to Administer >> Organic Groups >> Organic Groups Content Permissions (admin/og/content-permissions)
   and select which OG content node types should have access controlled by group administrators.
5. Go to group home page, a menu tab should appear for "content settings" in which group administrators can
   control which content fields are accessible to group members per activated node type.
